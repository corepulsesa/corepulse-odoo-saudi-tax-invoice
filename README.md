# corepulse-odoo-saudi-tax-invoice
Saudi Tax Invoice — ZATCA Phase 2 | Bilingual | SAR Symbol | Custom Header &amp; Footer

The problem this solves: Most Odoo invoice templates either lack the official Saudi Riyal symbol, require special font installations, or don't support custom branded headers and footers. This module solves all three.
What you get:
•	✅ ZATCA Phase 2 compliant — all mandatory fields, seller/buyer VAT, QR code
•	✅ Official SAR (﷼) currency symbol rendered as an image — works on any server, any OS, no font setup
•	✅ Full bilingual layout — Arabic (RTL) and English on every label
•	✅ Custom header & footer — upload your own branded image per company directly from Odoo settings
•	✅ Clean module install — zero Odoo core files modified, safe across upgrades
•	✅ Amount in words — both Arabic and English
